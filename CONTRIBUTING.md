# Contributing to PetSite

Thank you for your interest in contributing to PetSite! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/petsite.git`
3. Create a feature branch: `git checkout -b feature/your-feature`

## Development Setup

```bash
# Backend
cd backend
npm install
cp .env.example .env
npm run dev

# Web (in new terminal)
cd web
npm install
npm start

# Mobile (in new terminal)
cd mobile
npm install
npm run ios  # or npm run android
```

## Code Style

- Use camelCase for variables and functions
- Use PascalCase for components and classes
- Add comments for complex logic
- Follow existing code patterns

## Commit Messages

- Use clear, descriptive commit messages
- Start with an action verb: "Add", "Fix", "Update", etc.
- Example: `Add product filtering functionality`

## Pull Request Process

1. Update documentation if needed
2. Ensure all tests pass
3. Create a pull request with a clear description
4. Wait for review and address feedback

## Areas for Contribution

- Frontend UI improvements
- Backend API enhancements
- Bug fixes
- Documentation
- Testing
- Performance optimization

## Need Help?

- Check existing issues and documentation
- Ask in discussions
- Create an issue for bugs or suggestions

Happy coding! 🐾
